==================
Contributing guide
==================

Do you want to update features and documents?
You can contribute by your own works!!


Guidelines
==========

This is sub project of `atsphinx <https://atsphinx.github.io/>`_
and it is applied `atsphinx's standard contribution guide <https://atsphinx.github.io/en/contributing/>`_.

At first, please see it.

Set up workspace
================

I use these applications for this project's workspace.

+----------+------------------------------------+
| Name     | Description/Role                   |
+==========+====================================+
| Task     | | Task runner                      |
|          | | https://taskfile.dev/            |
+----------+------------------------------------+
| lefthook | | Git-hooks manager                |
|          | | https://lefthook.dev/            |
+----------+------------------------------------+
| uv       | | Python package manager           |
|          | | https://docs.astral.sh/uv/       |
+----------+------------------------------------+
| Typst    | | Generate demo and tools for test |
|          | | https://typst.app/               |
+----------+------------------------------------+

If you use `aqua <https://aquaproj.github.io/>`_, you don't have to install them manually.

.. code-block:: shell

   cp .env.example .env
   task setup-dev
